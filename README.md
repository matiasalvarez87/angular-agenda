angular-agenda
==============

## [Steps to setup the example](www.lta.com)

```
// Download the project
git clone git@github.com:matiasalvarez87/angular-agenda.git
cd angular-agenda

// Install dependencies
npm install
bower install

// Run the app
grunt
```
